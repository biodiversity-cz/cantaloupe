# Cantaloupe
No-root docker image of IIIF image server [Cantaloupe](https://cantaloupe-project.github.io/).

The base image is checked [once per month](.github/dependabot.yml#L6) for updates and all [included versions](.github/workflows/publish.yml#L16) of Cantaloupe are rebuild.
